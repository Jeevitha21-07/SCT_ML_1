# 🏠 House Price Predictor

An interactive **Machine Learning web application** that predicts house prices based on key features like area, bedrooms, bathrooms, quality, and garage capacity.

---

## 🚀 Live Demo

🌐https://sctml1-feqbqbqnmgayrjntebwxtk.streamlit.app/

---

## 📌 Features

* Predict house prices using **Linear Regression**
* Uses real-world dataset from Kaggle
* Interactive UI built with **Streamlit**
* Accepts user inputs dynamically
* Displays predicted price instantly
* Smart insights based on prediction

---

## 🧠 Machine Learning Model

* Algorithm: **Linear Regression**
* Features used:

  * GrLivArea (Square Footage)
  * BedroomAbvGr (Bedrooms)
  * FullBath (Bathrooms)
  * OverallQual (Quality)
  * GarageCars (Garage Capacity)
* Evaluation Metrics:

  * Mean Absolute Error (MAE)
  * R² Score
